AdminStudio 11
Application Readiness for Virtualization Suitability, Application Compatibility and Application Packaging Operations

» Install AdminStudio 11
» Learn More
» View Help Library
» Release Notes

AdminStudio Suite powers an enterprise's Application Readiness process for desktop transformations, migrations to Windows® 7, virtualization and day-to-day application packaging operations. AdminStudio Suite also removes a critical barrier to cloud adoption by automating the assessment, conversion and publishing of various virtual formats to both physical and cloud environments.

Its automation, integration with leading software distribution tools and workflow capabilities help companies streamline processes, leverage their investment in MSI packaged applications, publish physical and virtual software packages across hybrid environments from the client to the cloud and reduce the time and cost of application migration projects.

Trusted by more than 10,000 organizations, AdminStudio Suite eliminates the need for multiple standalone tools and provides an integrated solution for teams ranging from a single application packager to large distributed teams, reducing helpdesk calls and increasing user productivity by ensuring consistent and reliable application deployments.

 

New in AdminStudio 11

Detect, view and edit Microsoft Configuration Manager 2012 application metadata
Accurate identification of deployment metadata is crucial for populating Microsoft Configuration Manager 2012's new application model. AdminStudio 11 automatically gathers this metadata by mining package elements such as detection methods, requirements, and dependencies. All of this metadata, which you can easily view and update in AdminStudio, is transferred directly to Configuration Manager upon application publication.

Bulk Import from Microsoft Configuration Manager 2012 and 2007
AdminStudio 11 provides enhanced import functionality to help speed your migration to Microsoft System Center Configuration Manager 2012, easing your transition from its previous package-based model to its new application-based model. Bulk import eliminates the laborious and error-prone task of importing applications manually one at a time.

Server App-V Application Suitability Testing
AdminStudio 11 performs automatic suitability testing of packages for Microsoft Server App-V. By identifying conversion issues in advance, AdminStudio 11 allows users to focus first on those applications that lend themselves readily to the Server App-V environment. This speeds the conversion process and helps to accelerate the adoption of Server App-V technology.

Application Tagging
AdminStudio 11 now includes automatic ISO 19770-2 software tagging, which enables a more efficient software asset management and license optimization process. An application’s tag file contains accurate descriptive information (publisher, name, version, etc.) that can be collected and analyzed to provide an accurate application inventory.

Suite Installations via InstallShield
InstallShield 2012 Premier has been added into AdminStudio 11 Enterprise version, simplifying the configuration of multi-application software installations. Suiting helps IT eliminate problems and reduce support costs by ensuring that software installed manually uses the same installation process as software that is deployed automatically through a software delivery system.

Platform API
AdminStudio 11 includes a new platform API that enables users to sequence individual AdminStudio automated tasks to perform higher-level processes such as importing packages, checking for virtualization readiness, converting to virtualized packages, testing packages, distributing to Configuration Manager, and other custom tasks you define.
