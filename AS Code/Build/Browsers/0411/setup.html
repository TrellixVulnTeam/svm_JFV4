アプリケーション、アップデート、および修正プログラム (パッチ) の配布は、IT 部門における最も一般的かつ重要な機能です。 しかしながら現実は、これらのソフトウェア パッケージを配布する前段階で、その準備を適切に行えずにいる IT チームが数多く存在し、その結果、組織の生命線であるアプリケーションをクラッシュしたり、生産性の低下、およびヘルプ デスク費用の増加を引き起こすという問題が生じています。

アクレッソ ソフトウェアの AdminStudio 9.01 を使用すると、安定した従来型および仮想型ソフトウェア パッケージの配布を簡単に実現することができます。 AdminStudio では、パワフルなソフトウェアのパッケージ化テクノロジーに Web ベースのプロセス管理ツールが組み合わされているため、組織におけるデスクトップ管理費用の削減と IT 関連の生産性の増加が大幅に達成されます。

AdminStudio 9.01 の新機能
Microsoft Application Virtualization (App-V) のサポート: AdminStudio では、今回より Microsoft App-V 仮想化形式がサポートされています。 AdminStudio では、単一の Windows Installer パッケージを仮想パッケージに変換できるほか、複数の Windows Installer パッケージを単一の仮想パッケージ スイートに変換したり、多数の Windows Installer パッケージが含まれているディレクトリを個別の App-V アプリケーションに変換したりすることができます。

強化されたアプリケーションの仮想化のサポート: 変換前スキャンおよび PackageExpert の強化により、的確なデータおよび自動化プロセスの利用が可能になりました。これにより、既存の業務を利用して、アプリケーションの仮想化への移行をより効率的に計画することができるようになりました。

強化された Microsoft® Windows Vista™ のサポート: AdminStudio に搭載されている Vista のテストとパッケージ編集ツールを利用して、すべてのパッケージを Vista ガイドラインに準拠させることができます。

CMDB および ITIL の実装のサポート: AdminStudio では、CMDB で、ITIL ベースの Definitive Software Library (DSL) コンテンツをサポートする集中アプリケーション カタログが提供されています。 また、今回より、アプリケーション カタログで統合されているパッケージ データ、および、併合された CMDB ソースを含む、エンタープライズ データ ソースからリアルタイムに引き出された情報に基づいてレポートを作成する統合レポート機能が追加されました。

強化された自動化されたテスト機能: AdminStudio では、従来型または仮想型にかかわらず、ビルドされた個々のパッケージのテストにおいて、環境別ベスト プラクティス テスト シナリオが使用されるため、常に品質の高い出力が約束されます。
