On this page you can view the status of distributed applications, modify settings of distributed packages, and delete an existing distribution.

Package Administration Page Options (cont.)

Option

Description

SMS Package ID

Package ID assigned to this package by SMS.

Package Name

Name of the package.

Version

Version of the package.

Language

Language that the package was written for.

Description

Description of package.

Status

Lists the distribution status of the package: Distributed or Not Distributed.

Application Name

Name of application.

Company Name

Manufacturer of application.

Modify Settings

Click to modify the selected package's distribution settings. After you connect to the appropriate SMS Server, the SMS Package Configuration Page appears, where you can modify the distribution settings.

View Status

Click to view the status of a distributed application. SMS Web Console connects to the SMS Server, retrieves the status of that application, and displays that status as a text message.

Delete Distribution

Click to delete the selected application from SMS Web Console and/or the SMS Server.
