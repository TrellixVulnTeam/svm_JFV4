This dialog appears when you are attempting to connect to an SMS Server.

All of the settings configured on this page only apply to the selected package. The default settings (specified on the SMS Distribution Settings Page) do not change.

SMS Server Connection Settings Page Options

Option

Description

Site Server

Enter the machine name on which SMS Server is installed. (This could be a local or remote machine.) When you make a selection in this field, the Site Code field is populated with a list of sites available on the SMS server.

Site Code

Select the SMS site you want to use for software distribution.

User Name

Enter a valid user name for the SMS Server you want to connect to, such as: Company\UserName.

Password

Enter the appropriate password.

Next

Click to proceed with the connection.
