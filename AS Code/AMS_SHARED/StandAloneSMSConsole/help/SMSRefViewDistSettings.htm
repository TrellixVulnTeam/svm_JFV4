On the SMS Distribution Settings Page, which opens when you click Distribution Settings on the SMS Web Console Home page, you set the default SMS Server connection settings.

SMS Distribution Settings Page Options (cont.)

Option

Description

Site Server Name

Enter the machine name on which SMS Server is installed. (This could be a local or remote machine.) When you make a selection in this field, the Site Code field is populated with a list of sites available on the SMS server.

Site Code

Select the SMS site you want to use for software distribution.

User Name

Enter the user name appropriate for the SMS Site you want to connect to.

Update

Click to save you entries.

Note

The values on entered on this page will be used as the default values for any new package you target for distribution. However, you can change these values while configuring the package.
