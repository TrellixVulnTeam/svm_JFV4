Configuring Distribution Settings

You configure the SMS Server connection settings on the SMS Distribution Settings Page.

Note

The values on entered on the SMS Distribution Settings Page will be used as the default values for any new package you target for distribution. However, you can change these values while configuring the package.

Task

To configure distribution settings:

  1. Launch SMS Web Console. The SMS Web Console Home Page opens.
  2. Click Distribution Settings. The SMS Distribution Settings Page opens.
  3. In the Site Server Name field, enter the machine name on which SMS Server is installed. (This could be a local or remote machine.) When you make a selection in this field, the Site Code field is populated with a list of sites available on the SMS server.
  4. In the Site Code field, select the SMS site you want to use for software distribution.
  5. In the User Name field, enter the user name of a user who has access to the SMS Server.
  6. Click Update to save your entries
  7. Click the Home link to return to the SMS Web Console Home Page.
