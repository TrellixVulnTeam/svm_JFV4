SMS Web Console Home Page

AdminStudio Enterprise Server 8.0 » SMS Web Console

The Software Distribution Page is the “home” page of SMS Web Console. From this page you can choose to distribute a new package, perform package administration, and configure distribution settings.

SMS Web Console Home Page Options (cont.)

Option

Description

Distribute a New Package

Click to access the Distribute Package Page where you can select a package to distribute using distribution providers.

Package Administration

Click to access the Package Administration Page, where you can view the status of distributed applications, modify settings of distributed packages, and delete an existing distribution.

Distribution Settings

Click to access the SMS Distribution Settings Page, where you can configure global distribution settings or configure distribution provider settings.
