Modifying the Distribution Settings of a Package

AdminStudio Enterprise Server 8.0 » SMS Web Console

Click Package Administration on the SMS Web Console Home Page to see the status of a distributed package, delete a package, or modify a package’s distribution settings.

Task

To modify the distribution settings of a package:

  1. Launch SMS Web Console. The SMS Web Console Home Page opens.
  2. Click Package Administration. The Package Administration Page appears, listing all packages which are targeted for distribution.
  3. Select a package and click Modify Settings. The SMS Server Connection Settings Page appears.

    4. If this package has not been distributed, you can change the Site Server and Site Code fields on the SMS Server Connection Settings Page. However, if the package has been distributed, you cannot change these fields.

  4. Enter a valid User Name and Password for the SMS Server you want to connect to, and click Next. The General tab of the Package Settings View of the SMS Package Configuration Page opens.
  5. Edit the package's distribution settings as described in Setting Package Configuration Options.
