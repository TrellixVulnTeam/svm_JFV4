AdminStudio Enterprise Server is directly integrated with Microsoft Systems Management Server (SMS) 2003 via the SMS Web Console, which enables enterprises using the SMS distribution system to cut down software rollout times by greatly simplifying their distribution efforts enterprise-wide.

You can use SMS Web Console to prepare, publish, and distribute applications through your SMS system without ever touching the SMS server console, helping you improve your application management efforts while simplifying your overall SMS management requirements. You can use SMS Web Console to perform all of your SMS administrative tasks, including publishing and advertising applications and retiring or deleting applications from circulation. SMS Web Console also allows the use of existing SMS collections to distribute your applications, which simplifies targeting.

Using the centrally managed SMS Web Console, you can better monitor any application's SMS deployment success using a single interface, simplifying the process and saving you time.
