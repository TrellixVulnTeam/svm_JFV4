Using SMS Web Console to Distribute Packages

AdminStudio Enterprise Server 8.0 Â» SMS Web Console

You can use SMS Web Console, a Web-based application, to configure package distribution options and distribute setup packages without going to SMS Server. By using the SMS Web Console, you can:

  • Create and distribute packages to SMS Server from SMS Web Console.
  • View the package deployment status from SMS Web Console after it is targeted to distribute using SMS.
  • Choose the target audience for deployment by browsing through the SMS collections.
  • Configure program options for packages and target multiple distribution points for packages.
  • Refresh source files on the distribution points if source files are changed.
  • Create access accounts for setup packages to secure source files.
  • Delete the packages.

Caution

SMS Web Console features cannot be installed on 64 bit machines because Microsoft .NET Framework 2.0 does not support 64 bit machines.
