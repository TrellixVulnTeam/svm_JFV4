SMSPackageID: SMS Package ID given by SMS

SMS_PackageRootInfo_SelectedIndex = To store the selected tab index In the packagerootinfo.ascx
SMS_ProgramInfo_SelectedIndex = To store the selected tab index in the programinfo.ascx
SMS_AdvtInfo_SelectedIndex = To store the selected tab index in the Advertisementinfo.ascx

SMSSiteName = sms site name
SMSSitecode = sms site code
